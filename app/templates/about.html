MuseumWiki - Collection d'œuvres d'art

Le projet

MuseumWiki est une base de données d'œuvres d'art provenant de Wikidata. Elle est mise à jour régulièrement pour vous offrir un aperçu des plus grandes œuvres des musées du monde entier.

Sources

Les données sont extraites de Wikidata via des requêtes SPARQL et concernent principalement les œuvres de grands maîtres comme Monet, Picasso, Van Gogh, et bien d'autres.

Technologies

  • Flask - Framework web Python
  • Pandas - Traitement des données
  • Bootstrap - Interface utilisateur
  • Wikidata - Source des données

Contact

Ce projet est développé et maintenu par un passionné d'art et de données.
